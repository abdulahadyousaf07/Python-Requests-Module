## 🛠 Requirements

* Python 3.x
* requests library

Install the library using:

```bash
pip install requests
```

## ▶️ Running the Code

Clone the repository:

```bash
git clone <your-repository-url>
```

Move into the project directory:

```bash
cd requests
```

Run any Python file:

```bash
python filename.py
```
